# BioSeq Toolkit
BioSeq Toolkit is a collection of python scripts that performs various DNA/RNA/Protein analysis.
## Current Features

1) DNA nucleotide counter
     Counts the number of A, T, G, and C nucleotides.
2) Reverse DNA sequence
     Produces the sequence in reverse order.
3) DNA complement
     Converts each nucleotide to its complementary base (A↔T, C↔G).
4) Reverse complement
     Computes the reverse complement by combining reverse and complement.
5) Longest DNA sequence finder
     Finds the longest sequence from a list of DNA sequences.
6) Codon count
     Returns len(DNA) / 3 as the number of codons.
7) Palindrome check
     Checks whether a sequence is identical to its reverse.
8) Hamming distance
     Counts the number of differing positions between two equal-length DNA sequences.
9) Motif search
     Attempts to search for occurrences of a motif in a DNA sequence.
10) RNA → DNA conversion
     Replaces every U with T.
